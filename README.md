Task Objective
Perform exploratory data analysis (EDA) on the Iris dataset to understand feature distributions, relationships, and patterns among species.
Dataset Used
Name: Iris dataset (Seaborn)
Samples: 150
Features: sepal_length, sepal_width, petal_length, petal_width
Target: species (setosa, versicolor, virginica)
Models Applied
No predictive models; analysis focused on descriptive statistics and visualizations.
Key Findings
Feature Insights: Petal dimensions vary more distinctly across species than sepal dimensions.
Visual Analysis:
Scatter plots show setosa is easily separable, while versicolor and virginica overlap.
Box plots highlight feature ranges and potential outliers.
Conclusion: Petal length and width are strong discriminators for species classification.
